Overview

Psych parser wrapper

Wraps Psych.parse_stream to provide TreeHaver-compatible parsing.

Examples:

parser = TreeHaver::Backends::Psych::Parser.new
parser.language = Language.yaml
tree = parser.parse(yaml_source)
